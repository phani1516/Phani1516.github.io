QuickDrop

Browser-to-browser P2P file sharing across devices on the same network — no uploads, no accounts, no cloud. A self-hosted Snapdrop alternative with a glass UI and an animated radar.

Why it matters → Pure systems engineering, no AI shortcuts available: chunked 16KB transfers with `bufferedAmountLowThreshold` flow control, STUN/TURN ICE handling for symmetric NAT traversal, IP-based rate limiting. Server only relays signaling — every byte of every file flows directly between browsers, encrypted.
